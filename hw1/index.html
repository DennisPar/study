упорядоченный список,5 парных тегов.
  1. Тег-div блочный элемент.
  2. Тег-head предназначен для хранения служебной информации о странице.
  3. Тег-html является обязательным, так как содержит всю структуру документа и является оболочкой для остальных элементов.
  4. Тег-title Определяет основной заголовок web - страницы.
  5. Тег-p Определяет текстовый абзац.
неупорядоченный список,5 одиночных тегов.
  • тег br (break row) обрывает поток текста и вставляет перенос строки
  • тег input - является пустым элементом и содержит только атрибуты
  • тег param используется для определения параметров плагинов для элемента object.
  • тег hr (horizontal rule) определяет горизонтальную линию, которая вставляет разрыв строки до и после себя и занимает всю доступную ширину.
  • тег img (img сокращение от англ. слова image - изображение).

Что такое верстка

Верстка по праву считается одним из самых важных этапов создания сайта . От
качества работы и подготовки, знаний специалиста зависит вся работа, качество
сайта, его вид и функционирования. Даже незначительная ошибка, допущенная в процессе
верстки, может привести к катастрофическим последствиям.
Неправильный код может дать серьезную нагрузку на сайт, неправильно отобразить
компоненты, в результате чего:

  1. посещаемость сайта упадет.
  2. прибыль клиента будет стремиться к нулю.
